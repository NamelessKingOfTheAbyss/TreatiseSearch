# 全体概要

パワーエレクトロニクスとは、電力の変化や制御を効率的に行う技術だ。このパワーエレクトロニクス関連の設計には専門知識と時間が必要である。例えば、基盤となる電力変換器の回路設計には、放熱、ノイズ除去、電力、動作周波数などに関する高度な専門知識とノウハウがもとめられる。この開発のために膨大な時間とコストが、再生エネルギーやEVの普及を妨げていた。

本論文では、この課題に対し、USPMという新しい電力制御モジュールを開発・適用するアプローチをとっている。

* USPM(Universal Smart Power Module)：直列または並列に接続して組み合わせるだけで、様々な仕様の電力変換システムを短時間で構築できるようにするというモジュール。

# 具体的な技術アプローチ

1. 機能のオールインワン化とモジュール接続

UPSMは、主回路、ゲートドライバ、EMIフィルタ、検出回路、冷却システム、高速コントローラなど、電力変換に必要なすべての基本コンポーネントを1つの標準化されたハードウェアに統合している。

これにより、複雑なハードウェア・ソフトウェア設計を省略し、モジュールの接続方法を変えるだけで、数百ワットから数キロワットまで、電圧や電流、電力定格の異なる多種多様なシステムを構築できる。

2. 超小型コアパワーモジュールの開発

USPMの汎用性を高めるには、心臓部となるモジュールの小型化が必要だった。そこで、次世代半導体であるSiC-MOSFETを採用し、従来のIGBTモジュールの約1/3の体積となる超小型コアパワーモジュールを開発した。これを実現するために、以下の１つの技術が用いられている。

* 低熱抵抗化(ADB技術)：モジュールのドレイン側と銅回路パターンの雪像に、はんだではなく、AOB(原子拡散接合)をパワーモジュール分野で初めて適用し、熱抵抗を約33%削減し、10nH以下を達成した。
* 低インダクタンス化：ソース側をワイヤーボンディングではなく、パンプ(微小な突起)で直接接続することで、寄生インダクタンスを従来比で約80%削減し、10nH以下を達成した。

3. 無線通信による自律分散制御の配線と削減

数キロワット規模のシステムを作るためには、多数のUSPMを連結する必要があるが、マスターコントローラと各USPMを信号線でつなぐと、配線が複雑になり、システムの信頼性が落ちる。

これを解決するために、Bluetoothを用いた無線通信によってマスターコントローラと各モジュール間のやり取り(電圧や電流の指令値の送信)を行い、システム内の信号線を大幅に削減している。

4. ドループ制御によるシステム動作の安定化

複数のUSPMを接続して独立稼働させると、それぞれの制御が干渉しあってシステムが不安定になる。例えば、直列世俗時に電流制御が不安定になる、並列接続時に電圧の不均衡が起きるなど。

これを防ぐために、ドループ制御を適用している。実験により、この制御を用いることで電圧の不均衡が補正され、ゆがみのないきれいな正弦波が得られることが確認されている。

* ドループ制御：負荷に応じて電圧や電流を調整し、モジュール間で負荷を均等に分担させる制御
