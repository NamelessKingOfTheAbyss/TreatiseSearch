# 論文の全体像

本研究は、DWPTの実用化に向けた研究である。DWPTとは、Dynamic Wireless Power Transfer(走行中非接触給電)のことで、道路に埋め込んだ送電コイルから走行中のEVの受電コイルへワイヤレスで電力を送る技術である。EVの航続距離や充電時間の課題を根本から解決すると期待されている。

この給電には、磁界共鳴方式が用いられていて、特にコイル間のズレに強いS-S方式が想定されている。DWPTでは、車がコイルの上を高速で通過するため、限られた一瞬で大電力を送る必要がある。しかし、急に送電を開始すると、過電流が発生してしまう。この過電流の整定時間を短縮することが大きな課題となっている。

* EV：Electric Vehicle：電気自動車。バッテリーに蓄えた電気でモーターを回して走る車
* 磁界共鳴方式：送電側と受電側のコイルを同じ周波数で共振させ、磁界を介して効率よく離れた場所へ電力を送る方式
* S-S方式：Series-Series：送電側・受電側ともにコイルと直列に共振コンデンサを接続する回路構成
* 整定時間：変化が起きてから安定した状態に落ち着くまでの時間

# 具体的な課題

従来、この過電流を防ぐための代表的な方法として、送電側のインバータで位相シフトを用いて、電圧を徐々に上げていくアプローチがとられていた。

しかし、この従来の手法には、定常値に達するまでに数msかかってしまうという課題があった。DWPTのように車が一瞬で通り過ぎる環境において、整定時間に数msもかかってしまうことは、充電できる時間と受け取れる電力量が大きく減ってしまうことを意味している。

したがって、「過電流の抑制」と「高速な整定」の両立が大きな壁となっている。

* 位相シフト制御：インバータ内のスイッチの切り替えタイミングをずらすことで、出力する電圧をコントロールする制御方法
* 定常値：電流の安定値

# 課題に対する解決アプローチ

筆者は、送電側ではなく、受電側、すなわち車側を制御するというアプローチを提案した。

まず、複雑な高周波回路の振る舞いを分析するために、包絡線モデルを導出した。これにより、回路全体を扱いやすい二次遅れ系として扱うことが可能になった。

* 包絡線モデル：交流波形の振幅の頂点を結んだ輪郭の動きを数式化し、システムをシンプルに表したモデル
* 二次遅れ系：入力に対して出力が振動しながら遅れて応答するシステム

そして、受電側にアクティブ整流器を搭載した。ここでのポイントは、波の重ね合わせの利用である。送電側から電力が送られてきたときに発生する振動と、受電側のスイッチをオンにした瞬間に発生する振動が、ちょうど逆位相になる絶妙なタイミングを計算し、そのタイミングで受電側のスイッチを切り替える。これにより、波同士が打ち消しあい、過電流の波を完全に消し去ることに成功した。

さらに、この手法を様々な状況で扱えるように、ダイオードの非導通モードを意図的に作り出す工夫を行った。スイッチングを細かく制御することで見せかけ上の電圧を調整し、波を打ち消すための電圧制約条件を緩和するアプローチも開発した。

* アクティブ整流器：半導体スイッチを用いて、電流を一方向に流すだけでなく、電圧をかけるタイミングも自在に制御できる装置
* 波の重ね合わせ：複数の波がぶつかったときに、それぞれの波が足しあわされる性質
* 逆位相：波の山と谷がちょうど逆転した状態
* 非導通モード：受電側の電圧が送電側からの誘起電圧よりも高く、電流が全く流れない状態
* 電圧制約条件：提案されたその手法を完全に機能させるために必要な受電側電圧の条件

# 提案されたアプローチの試行結果

提案手法を実際の回路や走行ベンチ(時速10kmや100kmを模擬した環境)で実験・シミュレーションした結果、驚異的な成果が得られた。

まず、従来法では数msかかっていた整定時間を、わずかµsオーダーにまで劇的に短縮しつつ、過電流をほぼ0%（実験値で3%など）に抑え込むことに成功した。整定時間が短縮された結果、最初の3ms間で受け取れる受電エネルギーは、従来の手法と比べて約44%も増加し、一瞬の通過でもより多くの電力を安全に受け取れることが実証された。

また、実際の道路では車の位置がズレるなどの変動が起きるが、これに対するロバスト性も検証された。受電電圧やスイッチングのタイミングが理想からズレた場合でも、従来法と比べて十分に過電流を抑え込めることが確認された。

さらに付録として、DWPTで必須となる「車がコイルの上に来たことの検知」において、車側から微小な検知パルスを出し、センサーレスでお互いを検知し、送電タイミングを合わせる手法との統合実験にも成功している。これにより、システム全体として安全かつ超高速なワイヤレス充電が可能であることが示された。
