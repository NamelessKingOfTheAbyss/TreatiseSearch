# 系統連系型PUC太陽光発電（PV）インバータのための強化学習ベースのコントローラ

本論文は、系統連系型5レベルPacked-U-Cell (PUC) 太陽光発電インバータにおいて、**強化学習（PPOアルゴリズム）を用いたモデルフリーな制御手法**を提案し、動的な環境下での**低高調波歪みの維持、キャパシタ電圧の安定化、および高い堅牢性**を実現できることをシミュレーションにより実証したものです。

## 概要

### 1. タイトルと著者
*   **タイトル**: 系統連系型PUC太陽光発電インバータのための強化学習ベースのコントローラ
*   **著者**: Alamera Nouran Alquennah（テキサスA&M大学）、Melanie Chida（テキサスA&M大学）、Tassneem Zamzam（テキサスA&M大学）、Mohamed Trabelsi（クウェート科学技術大学）

### 2. 概要 (Abstract)
この論文は、系統連系型太陽光発電（PV）システムに使用される**Packed-U-Cell (PUC) インバータ**のための、**強化学習（RL）**を用いた新しい制御手法を提案しています。
主な目的は、PVの最大電力点（MPP）に応じて変化する参照電流を追跡しながら、補助キャパシタの電圧を一定に保ち、生成される電流の**全高調波歪み（THD）を最小化**することです。本研究では、Actor-Criticアーキテクチャと**近接方策最適化（PPO）**アルゴリズムを採用しています。シミュレーションの結果、電圧誤差は最大1.9Vに抑えられ、THDは2%〜4.5%と良好な値を示しました。また、回路素子（キャパシタやインダクタ）のサイズ変更に対しても高い**堅牢性**を示しています。

### 3. はじめに (Introduction)
*   **背景**: 世界的なPV発電の増加に伴い、高効率なインバータの需要が高まっています。**マルチレベルインバータ（MLI）**は、従来の2レベルインバータと比較して、高画質な信号生成、低高調波、電磁干渉の低減、デバイスへの電圧ストレス緩和といった利点があります。
*   **PUCトポロジー**: PUCは、単一のDC電源、1つのフライングキャパシタ、6つのスイッチで構成され、部品数を抑えつつ5または7レベルの電圧を出力できる優れたトポロジーです。
*   **制御の課題**: 系統連系システムでは、THDを5%未満に抑える（IEEE 929-2000規格）必要がありますが、従来のモデルベース制御（SMCやMPC）は正確な数学的モデルが必要であり、計算負荷も高いという課題があります。
*   **強化学習の利点**: RLベースのコントローラは、システムの**モデル情報を必要とせず**、環境との相互作用を通じて最適な制御ポリシーを学習するため、動的な変化に強く、計算負荷も軽減できます。

### 4. PUCトポロジーとモデリング (PUC Topology and Modelling)
*   **構成**: 研究対象のPUC-MLIは、2つのキャパシタ（$C_1$, $C_2$）を介して接続された3つのセルで構成されます。
*   **電圧制御**: $C_1$の電圧はDC-DCコンバータによってMPPTアルゴリズムに基づき制御されます。一方、5レベルの電圧出力を得るためには、$C_2$の電圧を参照値（$V_1/2$）に維持する必要があります。
*   **数式モデル**: システムの状態は、キャパシタ電圧の微分方程式と、グリッドに注入される電流の微分方程式によって記述されます。

### 5. 強化学習ベースの制御設計 (RL Based Control Design)
コントローラ（エージェント）の役割は、各タイムステップで適切なスイッチング構成を選択し、以下の目標を達成することです。
1.  **電流追跡**: THDを最小化するために参照電流信号を追跡する。
2.  **電圧調整**: キャパシタ電圧$V_2$を参照値付近に維持する。

*   **アクション空間 (Action Space)**: 8つの離散的なスイッチング構成から選択します。
*   **状態空間 (State Space)**: $V_1$、$V_2$、グリッド電流、グリッド電圧、参照電流などの6次元の連続空間です。
*   **報酬関数 (Reward Function)**: 電流誤差と電圧誤差、およびそれらが制限を超えた場合のペナルティに基づいて設計されており、エージェントはこの**報酬を最大化**するように学習します。
*   **アルゴリズム**: Actorネットワーク（ReLU活性化関数）とCriticネットワーク（tanh活性化関数）を組み合わせた**PPOアルゴリズム**を使用しています。

### 6. シミュレーション結果と考察 (Simulation Results and Discussion)
テキサス州の気象データを用いた30分間のシミュレーションにより、提案手法の有効性が検証されました。
*   **電圧および電流制御**: 学習済みエージェントは、PV電圧の変化に合わせて$V_2$電圧を正確に調整し、グリッド電流を参照信号に同期させることに成功しました。
*   **電力品質**: 生成された電流のTHDは、電流ピーク値に応じて2.32%（7.7A時）や3.56%（5.2A時）となり、**5%未満という基準を十分にクリア**しました。
*   **堅牢性**: キャパシタ容量を100µFから2000µFまで変化させても、電圧誤差やTHDへの影響は軽微であり、高い堅牢性が確認されました。
*   **比較**: 従来のPI制御やMPC、SMCと比較して、本手法は特定の条件下でSMCと同等のTHD性能を示しつつ、モデルフリーであるという利点を持っています。

### 7. 結論 (Conclusion)
本論文では、系統連系型PUC 5レベルインバータのためのRLベースのコントローラを提案しました。PPOアルゴリズムを用いたこのコントローラは、PVの動的な出力変化に対応しながら、**低THDでの電流供給、統一力率での動作、キャパシタ電圧の安定維持**を達成しました。特に、物理モデルに依存しないモデルフリーな特性により、回路パラメータの変動に対しても極めて高い堅牢性を示すことが実証されました。
