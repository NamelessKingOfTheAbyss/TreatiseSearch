# 全体像

インバータとは、直流電力を交流電力に変換し、電力網へ送り出す装置である。インバータの電流制御にはこれまでPI制御が用いられてきた。

* PI制御(Proportional-Integral Control)：現在の出力と目標値とのずれを計算し、それに比例した操作と過去の誤差の蓄積をもとに修正を加える制御方式。産業界で最も一般的な制御方式

しかし、PI制御はモデル依存性が高いという欠点があった。モデル依存性とは、PI制御における最適なパラメータを設定するために、対象となるインバータや電力網の抵抗値やインダクタンスといった正確な数学モデルをあらかじめ知っておかなければならない、ということである。モデルに依存しているため、温度変化や電力網の変動などがあると、設定が合わなくなり、性能が落ちてしまう。

この課題を解決するために、論文では強化学習RL-DDPGを用いた新しい制御器を提案している。

* DDPG(深層決定論的方策勾配法：Deep Deterministic Policy Gradient)：連続的な数値をコントロールするのに適したAIの学習アルゴリズム。最大の特徴は、モデルフリーであることで、事前の複雑な数式モデルを必要としない。また、環境との相互作用・試行錯誤からオンラインで直接学習することができる。
* アクタークリティック構造：DDPGが採用しているAIの頭脳構造。AIの中に2つのニューラルネットワークを持つ。アクターは、現在の状態を見て、次にどう操作するかを決定する。クリティックは、アクターがとった行動がどれくらい良かったかを評価し、アクターに修正値をアドバイスする。

# 具体的な実装

AIが学習する上で、複雑な電気の動きをシンプルにする工夫がなされている。

1. d-q座標変換：波のように激しくプラスとマイナスを行き来する三相交流電流を、数学的に直流のように変換すること。これにより、d軸成分を有効電力、q軸を無効電力として分けてAIに入力でき、制御が格段に簡単になる
2. 報酬関数(Reward Function)：AIに、「何が成功で、何が失敗かを数えるための採点基準。この論文では、「出力電流のズレ（誤差）が小さいこと」と「無駄に操作を急変させないこと（過去の操作からの変化量が小さいこと）」の両方を満たした時に、AIに高いスコア（報酬）を与えるように数式が組まれている。

# シミュレーション結果

提案されたRL-DDPG制御器をPI制御器と比較した結果、以下の2点で劇的な向上が見られた。

* THD(全高調波歪：Total Harmonic Distortion)：作り出された交流の波形が、理想的なきれいな波からどれくらい歪んでいるかを示す指標。数値が低いほど高品質な電気。
* 定常特性と動的応答(過渡応答)：定常特性は、目標値に達した後の安定性のこと。動的応答は、目標値が急激に変化したときに、どれだけ素早く正確に追従できるかを示す能力。

THDにおいては、PI制御では1.41%だったのに対し、DDPG制御では0.74%となり、ゆがみを焼く47%も低減させた。

また、PI制御は動的応答を良くしようとすると、オーバーシュートが35%も発生していたが、DDPG制御では、これをわずか8%に抑え込んだ。

* オーバーシュート：目標値が急激に変化したときにどれだけ素早く、正確に追従できるかを示す能力。
