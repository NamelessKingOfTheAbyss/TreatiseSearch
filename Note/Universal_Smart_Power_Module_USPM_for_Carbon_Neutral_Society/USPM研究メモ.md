# カーボンニュートラル社会に向けたユニバーサル・スマート・パワー・モジュール（USPM）

## 概要
この論文は、**カーボンニュートラル社会の実現に向け、再生可能エネルギーの変動に即応し、かつ専門知識がなくても電力変換システムを容易に構築できる「ユニバーサル・スマート・パワー・モジュール（USPM）」とその中核となる超小型コアパワーモジュールの概念および開発結果**をまとめたものです,,。

各節の概要は以下の通りです。

*   **I. はじめに（INTRODUCTION）**
    太陽光や風力などの再生可能エネルギーを主力電源化し、電力を安定供給するためには、次世代のエネルギーマネジメントシステムが不可欠であり、その鍵となる**パワーエレクトロニクス技術の普及を加速させるためにUSPMの研究開発**が行われています。
*   **II. ユニバーサル・スマート・パワー・モジュール（USPM）**
    USPMは、主回路、ゲートドライバ、EMIフィルタ、冷却システム、および**高速コントローラを一体化した多機能ビルディングブロック**であり、これらを直列または並列に接続するだけで、複雑な設計なしに多様な仕様のシステムを構成できます,。超高速フィードバック制御や、配線を削減するための**Bluetoothによる無線通信**を用いた自律分散制御などの要素技術が導入されています,。
*   **III. コアパワーモジュール（CORE POWER MODULE）**
    USPMの汎用性を支えるため、SiC-MOSFETを用いた**超小型コアパワーモジュール**が開発されました。原子拡散接合（ADB）による低熱抵抗化と、バンプ接続による低インダクタンス化（従来比80%減）を実現しており、全体的なボリュームは従来のIGBTモジュールの約1/3に抑えられています,,。
*   **IV. AC/AC電力変換システムへの適用例（EXAMPLE OF USPM SYSTEM FOR AC/AC POWER CONVERTER）**
    USPMを用いたAC/ACコンバータの構成例が示されており、複数のモジュールをISOP接続（入力直列・出力並列）した際の安定性を確保するため、**ドループ制御**が適用されています,。実験の結果、提案された制御手法によって電圧の不均衡が補正され、良好な正弦波形が得られることが確認されました,。
*   **V. 結論（Conclusion）**
    USPMは、高い応答性と汎用性を兼ね備え、設計ノウハウがなくても電力電子機器の開発を可能にする革新的な技術であり、**次世代エネルギーマネジメントシステムの発展とカーボンニュートラル社会の実現に大きく貢献する**ことが期待されます,,。

### 概要

太陽光や風力発電などの変動する電源に瞬時に対応し、汎用性とスマートさを兼ね備えることで、急速に拡大する再生可能エネルギー市場に完全対応可能な「ユニバーサル・スマート・パワー・モジュール（USPM）」のコンセプトと結果について説明する 。USPMの内蔵コントローラは、マスターコントローラから受信した指令値に対して高速かつ高精度に応答（フィードバック）できるため、変動電源の高速制御が可能である 。USPMは、電力変換主回路と高速コントローラに加え、ゲートドライバとEMIフィルタで構成されている 。これにより、パワーエレクトロニクス設計の簡素化を実現する 。従来の設計には多様なノウハウと専門スキルが必要であったが、USPMはバリエーションの異なるモジュールを接続するだけで構成できるため、設計が簡略化される 。また、USPMの特徴である汎用性を生み出すため、構成要素であるコア・パワー・モジュールは超小型化されている 。超小型化技術として、低熱抵抗化のためにSiC-MOSFETのドレイン側に原子拡散接合（ADB）を適用し、低インダクタンス化のためにソース側にバンプ技術を開発・適用した 。新たに開発されたコア・パワー・モジュールは、汎用的なSi-IGBTモジュールの約$1/3$の体積を実現している 。パワーエレクトロニクス製品の構築を容易にするUSPMと、それを可能にする超小型コア・パワー・モジュールは、カーボンニュートラル社会の実現に貢献する 。

### I. はじめに

現在、太陽光や風力発電などの再生可能エネルギーを主力電源として活用するための電力供給安定化技術、電気自動車（EV）の蓄電機能、およびそれらのエネルギーマネジメント技術の研究開発が急務となっている 。カーボンニュートラル社会の実現における基幹技術の一つがパワーエレクトロニクス技術であり、その用途は図1に示すように、小・中容量から大容量まで多岐にわたる 。このような電気エネルギー社会では、電力を最適に運用する次世代エネルギーマネジメントシステムの構築が求められている 。

このような状況下、内閣府の戦略的イノベーション創造プログラム（SIP）第2期（2018-2022年度）のテーマの一つである「IoE社会のエネルギーシステム」において、USPM（ユニバーサル・スマート・パワー・モジュール）の研究開発が行われている 。USPMは、高速・高精度でありながらパワーエレクトロニクス設計を簡素化する新しい電力制御装置である 。専門的な設計ノウハウがなくても、USPMの接続を変更するだけで設計を完了できる点が特徴である 。

再生可能エネルギーを主力電源とするには、電力変換器に高速コントローラと大容量蓄電池（またはEV用蓄電池）を組み合わせ、変動する電力に対して瞬時に安定化を図ることが重要である 。一般に、これらの電力変換器は用途ごとに最適設計されるが、回路設計には放熱、ノイズフィルタリング、電力、動作周波数などの専門知識が必要となる 。回路開発における試行錯誤に伴う膨大な時間とコストは、再生可能エネルギーやEV普及の妨げの一つとなっている 。したがって、設計の高速化が求められている 。

新たに開発されたUSPMは、高速・高精度な制御を可能にするだけでなく、設計を簡素化するため、IoE社会における次世代エネルギーマネジメントシステムの鍵となる技術と位置付けられている 。

### II. ユニバーサル・スマート・パワー・モジュール（USPM）

図2にUSPMの模式図を示す 。USPMは、数百ワットから数キロワットの定格電力を扱う多機能なビルディングブロックである 。現在、高い拡張性と製造コスト削減を目指し、同様のコンセプトに基づいたパワーエレクトロニクス・ビルディングブロック（PEBB）の研究が世界中で活発に行われている 。PEBBは、標準化されたハードウェア上に電力変換やゲートドライバなどの機能を統合し、設計を簡素化する 。しかし、従来のPEBBであっても、依然としてパワーエレクトロニクス設計には専門スキルが必要であった 。

USPMは、主回路、ゲートドライバ、EMIフィルタ、検出回路、冷却システム、および高速コントローラのすべての基本コンポーネントをモジュール内に統合している 。USPMコンセプトの目的は、設計を簡素化し、長いリードタイムなしでシステムを開発することにある 。USPMは、小容量の電力変換器として独立して動作する 。USPMシステムは、複雑なハード・ソフト設計なしに、接続のバリエーションのみでパワーエレクトロニクスシステムを実現する 。特に、電圧、電流、電力定格を含む多数の仕様は、USPMの直列または並列接続によって達成される 。

USPMの優れた機能を実現するために、以下の要素技術が開発された：

1. 数十MHzのサンプリング技術を利用した、モジュール内コントローラによる超高速フィードバック制御 。


2. 無線通信（Bluetooth）の適用による信号線の排除と、複数USPMの自律分散制御 。


3. アクティブゲートドライブ技術と高速コントローラによる、超高速WBG（ワイドバンドギャップ）パワーデバイスのスイッチング波形最適化 。


4. 直列または並列に接続された独立分散制御モジュールの動作を安定化させる新しい制御手法 。


5. 広帯域ノイズを減衰させる、アクティブ・パッシブ両方式を併用した新しいノイズフィルタ技術 。



図3にUSPMのシステム構成を示す 。このシステムは、複数のUSPMと、それらの動作を制御するマスターコントローラで構成される 。USPMは、DC-DC、DC-AC、AC/DC、AC/ACといったあらゆる電力変換システムをシンプルな構成で構築できる 。マスターコントローラはすべてのUSPMを制御し、フィードバック制御のための電圧・電流指令等の信号を各USPMの高速コントローラへ転送する 。

システムの信頼性を低下させる原因となる膨大な信号線を削減するため、マスターコントローラと各USPM間の通信にはBluetoothなどの無線通信が採用されている 。マスターコントローラは、通信遅延を考慮し、瞬時の物理情報ではなく実効値（RMS）などの定常状態情報を転送する 。

図4にUSPMの適用例（再生可能エネルギーと蓄電池を用いた電力貯蔵システム）を示す 。従来システムでは各変換器（DC/DC、DC/AC等）に専門的な設計が必要であったが、USPMを用いることでシステム設計は劇的に簡素化される 。

### III. コア・パワー・モジュール

USPMの構成要素であるコア・パワー・モジュールは、SiC、GaN、$Ga_{2}O_{3}$などのWBGデバイスの特性を最大限に引き出すため、低熱抵抗かつ低インダクタンス化を目指して開発された [cite: 106, 107][cite_start]。図5に、新たに開発されたコア・パワー・モジュール（SiC-MOSFET適用）と従来のIGBTモジュールの比較を示す [cite: 109][cite_start]。新モジュールの体積は従来比約$1/3$であり、USPMの構成に非常に適している 。

図6に、コア・パワー・モジュールの概要と適用技術を示す 。本モジュールは2in1構造（インバータの上下アームを構成）を持ち、容量は600V/100Aである 。以下の技術が適用されている：

*
**バンプ接続:** SiC-MOSFETのソース側と銅回路パターンの間をバンプで接続し、ワイヤボンディングを廃止することで低インダクタンス化を達成した 。


*
**原子拡散接合（ADB）:** ドレイン側と銅回路パターンの直接接合にADBを適用し、低熱抵抗化を実現した 。これはパワーモジュール分野で初の試みである 。



熱抵抗評価の結果、ADBによる接合は、はんだ接合と比較して熱抵抗（Rth）が約33%低いことが確認された（図7）。また、寄生インダクタンスは10nH以下を達成し、従来モジュールから約80%削減された（図10）。

---

### IV. AC/AC電力変換器へのUSPMシステム適用例

図11に、AC/AC電力変換システムへの適用例を示す 。このシステムでは、電流型、電圧型、および絶縁型のUSPMがISOP（入力直列・出力並列）接続されている 。各USPMは独立して動作するが、相互に干渉するため、制御を安定化させるために「ドループ制御」が適用されている 。

*
**電圧ドループ制御:** 電圧の不一致や検出ゲインの誤差による不安定化を防ぐため、AVR（自動電圧調整器）にインピーダンス特性を付与する 。


*
**電流ドループ制御:** USPMが直列接続された際の不安定化を解消するため、ACR（自動電流調整器）にアドミタンス特性を付与する 。



図14の実験結果によれば、提案するドループ制御を適用しない場合はDC電圧の不均衡により入力電流に歪みが生じるが、提案制御を適用することで電圧不均衡が補正され、純粋な正弦波形が得られることが確認された 。

### V. 結論

本稿では、汎用性とスマートさを兼ね備え、変動する再生可能エネルギーに即座に対応可能なUSPMのコンセプトと実験結果について説明した 。超小型コア・パワー・モジュールを実現するために、低インダクタンスのバンプ接続や低熱抵抗の原子拡散接合（ADB）といった新技術を開発した 。USPMは個別に高速コントローラを備えることでエッジ的な役割を果たし、電力供給を適切に制御できる 。また、多くの技術的ノウハウをモジュールに集約することで、専門知識がなくてもパワーエレクトロニクス装置を開発できる点は、大きな社会的インパクトを持つ 。USPMは、カーボンニュートラル社会を実現するための高度な次世代エネルギーマネジメントシステムへの貢献が期待される 。
