# 強化学習を用いた三相系統連系インバータ用制御器の設計

この論文は、従来の制御手法が持つモデル依存性やゲイン調整の複雑さを解消するため、**深層決定論的政策勾配（RL-DDPG）**アルゴリズムを用いた新しい制御器を提案しています。

---

### 1. Abstract（要旨）
本論文は、電圧形インバータ（VSI）用の制御器を**強化学習（RL）**を用いて設計する方法を提案しています。
*   **手法:** RL-DDPGアルゴリズムを採用し、アクター・クリティック構造によって、正確なシステムモデルの知識なしにオンラインで制御ゲインを更新します。
*   **成果:** 従来のPI（比例積分）制御と比較して、定常偏差、リプル、および全高調波歪（THD）を最小限に抑え、優れた電流追従性能を達成しました。

### 2. I. Introduction（序論）
再生可能エネルギーを系統（グリッド）に統合するためにVSIが広く使われていますが、その性能は電流制御手法の質に大きく依存します。
*   **従来の制御法の課題:** PI制御やPR（比例共振）制御はモデルパラメータに依存し、動作条件の変化に弱いという欠点があります。また、デッドビート制御はパラメータ変動に敏感で、ヒステリシス制御はスイッチング周波数が可変になるなどの問題があります。
*   **提案手法の利点:** 強化学習、特にDDPGを用いることで、環境との直接的な相互作用から学習し、モデルフリーかつオンラインで適応可能な制御を実現します。

### 3. II. System Modelling（システムモデリング）
制御器を設計・評価するために、系統連系三相VSIの動的モデルを構築しています。
*   **座標変換:** 三相交流（abc）変数を空間ベクトル理論を用いて静止座標系（$\alpha$-$\beta$）に変換し、さらに同期回転座標系（$d$-$q$）へ変換します。
*   **状態空間モデル:** インバータの特性を表す大信号モデルおよび、特定の動作点付近での挙動を示す小信号状態空間方程式を導出しています。これらの方程式により、d軸電流が有効電力を、q軸電流が無効電力を制御することが示されます。

### 4. III. Introduction to RL（強化学習の導入）
強化学習をマルコフ決定過程（MDP）の枠組みで説明しています。
*   **観測（Observation）:** 出力電流の$d$-$q$軸成分（$i_d, i_q$）とその基準値（$i_{dref}, i_{qref}$）をエージェントへの入力とします。
*   **報酬（Reward）:** 電流の追従誤差を最小化し、不要なアクションを抑制するための評価関数を定義します。
*   **エージェント（Agent）:** 観測をアクションにマッピングする「ポリシー（政策）」と、それを最適化する学習アルゴリズムで構成されます。

### 5. IV. DDPG Algorithm（DDPGアルゴリズム）
DDPGは、モデルを必要としないオンラインのアクター・クリティック型学習手法です。
*   **アクター・クリティック:** アクターがアクション（制御入力）を選択し、クリティックがそのアクションの質を評価して調整を促します。
*   **ニューラルネットワーク:** アクターとクリティックの両方に深層ニューラルネットワークを使用し、非線形なパターンを理解するためにReLU活性化関数を採用しています。これにより、パラメータへの依存を減らし、過学習を抑制します。

### 6. V. Simulation Verification（シミュレーション検証）
MATLAB/Simulinkを用いて、提案するRL-DDPG制御器と従来のPI制御器の性能を比較しています。
*   **定常性能:** RL-DDPG制御器は、PI制御器と比較して**THDを47%低減**しました（RL: 0.74% vs PI: 1.41%）。
*   **動的性能:** PI制御のオーバーシュートが35%であるのに対し、RL-DDPGは**8%に抑制**されており、過渡状態での変動も少ないことが確認されました。

### 7. VI. Conclusion（結論）
本論文は、三相VSIの制御においてRL-DDPGが極めて有効であることを示しました。
*   **結論:** 提案手法はモデルフリーであり、定常・動的両面でPI制御を凌駕する性能を発揮します。
*   **今後の課題:** 深層ニューラルネットワークを使用するため学習に時間を要することや、性能が報酬関数の設計に依存することが課題として挙げられています。
