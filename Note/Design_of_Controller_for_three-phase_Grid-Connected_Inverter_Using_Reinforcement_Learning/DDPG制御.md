# 強化学習を用いた三相系統連系インバータ用制御器の設計

この論文は、従来の制御手法が持つモデル依存性やゲイン調整の複雑さを解消するため、**深層決定論的政策勾配（RL-DDPG）**アルゴリズムを用いた新しい制御器を提案しています。

---

### 1. Abstract（要旨）
本論文は、電圧形インバータ（VSI）用の制御器を**強化学習（RL）**を用いて設計する方法を提案しています。
*   **手法:** RL-DDPGアルゴリズムを採用し、アクター・クリティック構造によって、正確なシステムモデルの知識なしにオンラインで制御ゲインを更新します。
*   **成果:** 従来のPI（比例積分）制御と比較して、定常偏差、リプル、および全高調波歪（THD）を最小限に抑え、優れた電流追従性能を達成しました。

### 2. I. Introduction（序論）
再生可能エネルギーを系統（グリッド）に統合するためにVSIが広く使われていますが、その性能は電流制御手法の質に大きく依存します。
*   **従来の制御法の課題:** PI制御やPR（比例共振）制御はモデルパラメータに依存し、動作条件の変化に弱いという欠点があります。また、デッドビート制御はパラメータ変動に敏感で、ヒステリシス制御はスイッチング周波数が可変になるなどの問題があります。
*   **提案手法の利点:** 強化学習、特にDDPGを用いることで、環境との直接的な相互作用から学習し、モデルフリーかつオンラインで適応可能な制御を実現します。

### 3. II. System Modelling（システムモデリング）
制御器を設計・評価するために、系統連系三相VSIの動的モデルを構築しています。
*   **座標変換:** 三相交流（abc）変数を空間ベクトル理論を用いて静止座標系（$\alpha$-$\beta$）に変換し、さらに同期回転座標系（$d$-$q$）へ変換します。
*   **状態空間モデル:** インバータの特性を表す大信号モデルおよび、特定の動作点付近での挙動を示す小信号状態空間方程式を導出しています。これらの方程式により、d軸電流が有効電力を、q軸電流が無効電力を制御することが示されます。

### 4. III. Introduction to RL（強化学習の導入）
強化学習をマルコフ決定過程（MDP）の枠組みで説明しています。
*   **観測（Observation）:** 出力電流の$d$-$q$軸成分（$i_d, i_q$）とその基準値（$i_{dref}, i_{qref}$）をエージェントへの入力とします。
*   **報酬（Reward）:** 電流の追従誤差を最小化し、不要なアクションを抑制するための評価関数を定義します。
*   **エージェント（Agent）:** 観測をアクションにマッピングする「ポリシー（政策）」と、それを最適化する学習アルゴリズムで構成されます。

### 5. IV. DDPG Algorithm（DDPGアルゴリズム）
DDPGは、モデルを必要としないオンラインのアクター・クリティック型学習手法です。
*   **アクター・クリティック:** アクターがアクション（制御入力）を選択し、クリティックがそのアクションの質を評価して調整を促します。
*   **ニューラルネットワーク:** アクターとクリティックの両方に深層ニューラルネットワークを使用し、非線形なパターンを理解するためにReLU活性化関数を採用しています。これにより、パラメータへの依存を減らし、過学習を抑制します。

### 6. V. Simulation Verification（シミュレーション検証）
MATLAB/Simulinkを用いて、提案するRL-DDPG制御器と従来のPI制御器の性能を比較しています。
*   **定常性能:** RL-DDPG制御器は、PI制御器と比較して**THDを47%低減**しました（RL: 0.74% vs PI: 1.41%）。
*   **動的性能:** PI制御のオーバーシュートが35%であるのに対し、RL-DDPGは**8%に抑制**されており、過渡状態での変動も少ないことが確認されました。

### 7. VI. Conclusion（結論）
本論文は、三相VSIの制御においてRL-DDPGが極めて有効であることを示しました。
*   **結論:** 提案手法はモデルフリーであり、定常・動的両面でPI制御を凌駕する性能を発揮します。
*   **今後の課題:** 深層ニューラルネットワークを使用するため学習に時間を要することや、性能が報酬関数の設計に依存することが課題として挙げられています。

## 単語

# 簡単な論文概要

インバータとは、直流電流を交流電力に変換し、電力網へ送り出す装置である。このインバータの制御は、これまでPI制御が広く使われてきた。

* PI制御：現在の出力と目標値とのずれを計算し、それを修正するように数学的に操作する制御方式。

PI制御において、最適なゲイン(パラメータ)を設定するためには、電力網の抵抗値やインダクタンスといった正確な数学モデルをあらかじめ知っておく必要がある。PI制御はこのモデル依存が高いため、温度変化や電力網の変動などのシステムパラメータが変化すると、設定が合わなくなり、性能が落ちてしまうという弱点がある。

このモデル依存性の課題を解決するために、新しく提案された制御手法が、RL-DDPGである。

* DDPG：連続的な数値をコントロールするのに適したAIの学習アルゴリズムで、モデルフリーなところが最大の利点である。モデルフリーとは、事前の複雑な数式モデルを必要とせず、環境との試行錯誤からオンラインで直接学習することができることである。

DDPGは、アクタークリティック構造を採用している。この構造は、AIの中にアクターとクリティックの２つのニューラルネットワークを持つ。アクターは、現在の状態を見て次にどう操作するかを決定する。クリティックは、アクターがとった行動がどれくらい良かったかを評価して、アクターにもっとこう修正した方がいいとアドバイスする。

そして、AI学習をさせるうえで、よりデータをシンプルにする工夫がなされている。具体的には、d-q座標変換と報酬関数が用いられている。

* d-q座標変換：複雑な3相交流を、回転子とともに回る直流の２軸に変換する技術。これにより交流信号を直流として扱えるため、制御が単純化される。d軸成分を有効電力、q軸成分を無効電力として分けてAIに入力する

* 報酬関数：何が成功で、何が失敗かを数えるための採用基準。出力電流の誤差が小さいことと、無駄に操作を急変させない（過去の操作からの変化量が小さい）ことの両方を満たしたときに、AIに高いスコアを与えるように数式に組まれている。

**つまり、DDPGという機械学習を利用して、モデルフリーに、PI制御より精度を向上することを見込んで、インバータの制御を試みようということ**

提案されたRL-DDPG制御器をPI制御器と比較した結果、以下の２点で劇的な向上が見られた。

* THD：THDとは、作り出された交流の波形が理想的なきれいな波（正弦波）からどれくらい歪んでいるかを示す指標である。数値が低いほど高品質な電気である。

* 定常特性と動的応答：定常特性とは、目標値に達した後の安定性のこと。動的応答とは、目標値が急激に変化したときにどれだけ素早く正確に追従できるかを示す能力。

THDについて、PI制御では1.41%だったのに対し、DDPGでは0.74%となり、ゆがみを47%も低減させた。また、PI制御は動的応答を良くしようとすると、目標を大きく飛び越えてしまう現象が35%発生していたが、DDPG制御はこれをわずか8%に抑え込んだ。

論文に記載されているシミュレーション結果および各制御手法の特性に基づき、PI制御とRL-DDPG制御の比較を表にまとめました。

| 比較項目 | 従来のPI制御 | 提案手法（RL-DDPG制御） |
| :--- | :--- | :--- |
| **システムモデルの依存性** | 高い（正確なシステムモデルのパラメータ知識が必要） | 不要（モデルフリーで環境との相互作用から学習） |
| **制御ゲインの調整** | 固定（動作条件が大きく変わると影響を受けやすい） | アクター・クリティック構造によりオンラインで自動・自律的に更新 |
| **定常性能と動的性能の両立** | 困難（一方を良くすると他方が悪化するトレードオフが生じやすい） | 両立可能（ニューラルネットワークによる非線形制御で最適化） |
| **THD（全高調波歪）**<br>※定常性能 | 1.41% | **0.74%**（PI制御より47%低減） |
| **オーバーシュート**<br>※動的性能 | 35% | **8%**（大幅に抑制） |
| **整定時間（Settling time）** | 0.025秒 | 0.028秒（PI制御とほぼ同等） |
| **過渡状態（急変時）の挙動** | 定常状態に達するまでにより多くのサイクルを要し、リプル（波打ち）や変動が大きい | 変動が少なく、より安定した電流波形を維持できる |
| **主な課題・デメリット** | システムパラメータの変動に弱い | 深層ニューラルネットワークを用いるため学習（トレーニング）に時間がかかる点や、性能が報酬関数の設計に大きく依存する点 |

このように、RL-DDPG制御はわずかに整定時間が長くなるものの、モデルに依存せずに定常性能（THDの低減）と動的性能（オーバーシュートの抑制）を高いレベルで両立させていることがわかります。
