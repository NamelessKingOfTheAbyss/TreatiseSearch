# ざっくりの概要

家の中の電気の通り道であるDCリンクには、太陽光パネルや蓄電池、電気自動車などがつながっている。ここで、複数の機械が同じ通り道につながっていると１つの機械が電気の量を変えたときに、通り道全体の電圧が揺れてしまう。これが原因で、つながっているほかの機械の動きまで邪魔されてしまい、システム全体がうまく動かなくなるという問題があった。

そこで、この論文では、機械同士がお互いの邪魔をしないようにするコントロール方法を発明し、紹介している。具体的には、全体の電圧を常にチェックして、もし揺れが起きそうになっても、それぞれの機械がその揺れをぴったりを打ち消すように自動で自分の動きを調整する。まるでノイズキャンセリングみたいに。

この方法を使うと、同じ通り道にいくつ機会をつないでも、ほかの機械から影響を受けなくなる。これにより、機械が急に動いても電気がぐらつかず、家全体の電気システムがとても素早く、安全に働くことが実験で証明された。

# 本論文の対象システム

本論文は、太陽光発電(PV)や蓄電池などを統合するホームエネルギー管理システム(EMS)などのDCマイクログリッドである。

* MIMO(他入力他出力)システム：電気回路に限らず、複数の入力と複数の出力が1本の同じ線を共有するシステム。本論文では、複数のコンバータが接続されたシステム全体を１つのMIMOシステムとして扱っている。

# 課題

従来のコンバータは、それぞれ独立したSISO(Single-Input Single-Output：１入力１出力)システムとしてPI制御を用いて設計されてきた。しかし、複数のコンバータが同じDCリンクを共有しているため、あるコンバータの電流制御がほかのコンバータの主力電圧や電流に影響を与えてしまう相互干渉が発生する。

これを伝達関数行列で表すと、目的の応答を示す対角項以外の非対角項がゼロにならない、すなわち、ほかの入力から影響を受けてしまうという問題があった。この干渉により、定電力負荷などが接続された際に、負のインピーダンス特性が生じ、システム全体が電圧不安定に陥る危険性があった。

* 伝達関数行列：入力から出力への応答を示す数式モデル

# 解決のための提案手法

論文では、動的電流非干渉制御(Dynamic Decoupling Current Control)を提案している。

まず、状態平均化法を用いて、システム全体の小信号モデル(微小な変動に対する応答モデル)を構築した。そのうえで、各コンバータの制御入力に対して、DCリンク電圧の変動分をリアルタイムで足し引きして相殺するというアプローチをとった。この操作を数式上で展開すると、伝達関数行列の非対象項が理論上完全にゼロになる。これにより、各コンバータの電流制御がほかのコンバータから完全に独立し、対象項は制御が極めて容易な「１次系」としてふるまうようになる。

* 状態平均化法(State-Space Averaging Method)：スイッチングの電源のオン・オフの細かい動作を１周期で平均化し、システムの動的なふるまいを連続的な数式として近似・モデル化する解析手法。

* １次系(First-order System)：1階の微分方程式で表される最もシンプルな動的システムのこと。入力に対して振動することなく、滑らかに目標値に追従する特性を持つ。

# 実証結果

提案された手法の有効性を証明するあt目、実機ベースのシステム同定とステップ応答実験を行った。

* 同定：実験データからモデルの特性を抽出すること

* 相対ゲインアレイ(RGA: Relative Gain Array)：MIMOシステムにおいて、どの入力と出力の結びつきが強く、どれくらい他のループと干渉しているかを定量的に評価する指標。干渉がない場合、目的のペアは0dB(完全な結びつき)、それ以外のペアは-∞ dB(結びつきゼロ)となる。

* 一般化ゲルシュゴリン帯を用いたナイキスト安定判別法：MIMOシステムでは、ほかの機械から干渉を受けると動きがブレてしまう。この「最悪どれくらいぶれるかという幅を帯として図に描く。図には不安定領域という絶対触れてはいけない領域がある。不安定領域に触れてしまうと、システムが暴走するリスクがあり、そこに触れていなければ、システムが安定していることが保証できる。

* ステップ応答実験：機械に対して「今すぐ数値を5から10に上げて！」のように、急な指示（ステップ入力）を出したときに、どれくらい正確に、かつ素早く動けるかを確かめるテスト。

1. 非干渉性能の確認

相対ゲインアレイの評価では、お互いに全く邪魔されず、高度に非干渉化できるようになったことが実証された。

2. MIMO安定性の保証

ナイキスト安全判別法で比較した。これまでのやり方では、動きのブレ幅である帯がおおきすぎて　危険な不安定領域を踏んでしまっていた。しかし、新しいやり方では、帯がとても小さくまとまっており、危険な領域をしっかりと回避していたため、安心して使えることが確認された。

3. 応答速度の向上

ステップ応答実験において、これまでのやり方では、勢い余って目標の数値を通り過ぎてしまうオーバーシュートという現象が起きていた。しかし、新しい手法では、オーバーシュートをしっかりと抑制し、しかも目標の数値にピタッと収束するまでの時間を大幅に短縮できることが確認された。
